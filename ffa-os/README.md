# FFA OS

Internal operating dashboard starter for Footy For All (FFA).

## Purpose
A lightweight control panel to:
- track sessions
- monitor revenue
- identify issues early
- prepare for automation

## Current Features
- static dashboard UI
- mock data layer
- basic alerts engine
- placeholder action buttons

## Structure

```text
ffa-os/
├── index.html
├── styles.css
├── app.js
├── data/
│   └── mock.json
└── README.md
```
